# Three Principles of Design  

This project demonstrates the three core principles of design: **Hierarchy**, **Contrast**, and **Rhythm**.  
The goal is to show how visual elements can guide user attention and improve interface comprehension.  

## Features  

- Animated background and page elements.  
- Responsive layout for different screen sizes.  
- Language switching (UA / EN).  
- Dynamic effects for cards and panels.  
- Lightweight version for mobile and low-performance devices.  

## Technologies  

- HTML5 / CSS3 / JavaScript  
- WordPress / Elementor (for integration)  
- Intersection Observer API for reveal animations  
- CSS Custom Properties for easy theming and style management
